# Togg App

## Run Locally

Clone the project

```bash
  git clone https://github.com/Batuhanseckin/togg_app.git
```

Go to the project directory

```bash
  cd togg_app
```

Install dependencies

```bash
  flutter pub get
```

Run project

```bash
  flutter run
```
